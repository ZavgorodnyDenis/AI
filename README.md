## 📊 Метод Ньютона — Генерация кода ИИ (Perplexity AI)
## Цель лабораторной работы
Исследовать применение генеративных моделей ИИ для создания кода численных методов решения нелинейных уравнений методом Ньютона. Оценить качество генерации на примере поиска √2 (уравнение x² - 2 = 0) с начальным приближением x₀=1.

##📋 Задачи
Сформулировать запрос к Perplexity AI для Python-реализации метода Ньютона

Протестировать на f(x)=x²-2=0 (корень √2≈1.414213562)

Сравнить с ручным расчетом

Верифицировать квадратичную сходимость
